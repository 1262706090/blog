# 汇编语言（第3版）


## 第 1 章 基础知识

**机器语言**：机器指令的集合。

**汇编语言**：由 3 类指令组成：

-   汇编指令：机器指令便于记忆的形式
-   伪指令
-   其他符号

CPU 与存储器之间的数据流动：

1. **存储器（Memory）**：通常指主存储器（RAM），是计算机系统中的主要数据存储区。它存放运行程序所需的指令和数据，CPU 从中读取或写入数据。

2. **寄存器（Register）**：寄存器是位于 CPU 内部的高速存储单元，能够临时存放指令、数据和操作数。寄存器的数据访问速度比主存储器快得多，但容量较小。

3. **数据流动过程**：
    - **指令和数据加载**：CPU 从主存储器中取出指令和数据，存放到寄存器中，以便进行高速运算。
    - **计算与处理**：CPU 在寄存器中对数据进行操作。寄存器与算术逻辑单元（ALU）直接配合完成算术或逻辑运算。
    - **结果存回**：运算结果会暂时存放在寄存器中，必要时再写回主存储器，以便后续指令使用。

因此，存储器确实存放了提供给 CPU 的数据，但为了加快运算速度，CPU 会将数据加载到内部寄存器中完成处理。
