# MIT-Missing-Semester


## 1. 课程概览与 shell

`< file`、`> file` 和`|`：

-   `< file`：将 file 的内容作为程序的输入（输入重定向）。
-   `> file`：将程序的输出写入到 file 中（输出重定向），如果文件已存在，它会被覆盖。
-   `|`：将一个程序的输出直接作为另一个程序的输入，实现程序之间的数据传递。

`echo`和`cat`：

-   `echo`：用于输出简单的文本或字符串。
-   `cat`：用于显示一个或多个文件的内容，并可以合并文件输出。

ls 命令可以查看文件，ls -l 命令中-l 代表长格式，这个选项会显示详细信息，可以帮助解决 Permission denied 问题。

在直接运行脚本时，需要执行权限；使用 source 则不需要。
